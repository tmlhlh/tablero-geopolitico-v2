# Tablero Geopolítico: Conceptos de Diseño

## Concepto 1: Análisis Estratégico Minimalista (Probabilidad: 0.08)

**Design Movement**: Modernismo Suizo + Data Visualization
**Core Principles**:
- Claridad radical: cada elemento comunica información crítica
- Jerarquía tipográfica fuerte: titulares en sans-serif bold, cuerpo legible
- Espaciado generoso: respira el contenido
- Monocromatismo con acentos: grises, negros, un color de alerta

**Color Philosophy**:
- Fondo: Blanco puro (confianza, neutralidad)
- Texto primario: Gris oscuro (legibilidad)
- Acentos: Rojo/Naranja (urgencia geopolítica), Azul (información)
- Propósito: Transmitir seriedad y análisis riguroso

**Layout Paradigm**:
- Grid asimétrico: columna ancha para contenido, columna estrecha para métricas
- Tarjetas flotantes con sombras sutiles
- Separadores verticales para dividir escenarios

**Signature Elements**:
- Líneas de tiempo horizontales para cada escenario
- Indicadores de riesgo tipo "medidor" (gauge)
- Iconografía geopolítica (banderas, símbolos de alianzas)

**Interaction Philosophy**:
- Transiciones suaves al cambiar escenarios
- Hover effects que revelan información adicional
- Animaciones que guían la lectura

**Animation**:
- Entrada: fade-in + slide de izquierda a derecha
- Cambio de escenario: cross-fade
- Indicadores: animación de "llenado" del gauge

**Typography System**:
- Display: Playfair Display (bold, elegante)
- Body: Inter (legible, neutral)
- Monospace: IBM Plex Mono (para datos/cifras)

## Concepto 2: Tablero Inteligencia Geopolítica (Probabilidad: 0.07)

**Design Movement**: Cyberpunk + Intelligence Dashboard
**Core Principles**:
- Sensación de "control center" de operaciones
- Información en capas: superficie visible, detalles ocultos
- Contraste alto: oscuro con acentos neon
- Dinamismo visual constante

**Color Philosophy**:
- Fondo: Gris muy oscuro/casi negro (inmersión)
- Acentos primarios: Verde neón (inteligencia, sistemas)
- Acentos secundarios: Cian/Magenta (datos, alertas)
- Propósito: Crear sensación de urgencia y sofisticación

**Layout Paradigm**:
- Paneles flotantes superpuestos
- Esquinas angulares, bordes con glitch effect
- Mapa central como eje, paneles laterales con datos

**Signature Elements**:
- Líneas de conexión entre actores geopolíticos
- Pulsos/animaciones en puntos críticos
- Código visual (números, símbolos) como decoración

**Interaction Philosophy**:
- Clics revelan capas de información
- Hover activa conexiones visuales
- Transiciones rápidas y dramáticas

**Animation**:
- Entrada: glitch + fade
- Datos: números que "cuentan" hacia el valor final
- Conexiones: líneas que se dibujan en tiempo real

**Typography System**:
- Display: Space Mono (monoespaciado, futurista)
- Body: Roboto Mono (técnico, legible)
- Acentos: IBM Plex Sans (títulos secundarios)

## Concepto 3: Prospectiva Académica Elegante (Probabilidad: 0.06)

**Design Movement**: Editorial Design + Academic Rigor
**Core Principles**:
- Presentación como publicación de investigación
- Tipografía sofisticada y jerarquía clara
- Uso de espacios en blanco para respiración
- Ilustraciones conceptuales sutiles

**Color Philosophy**:
- Fondo: Crema/Beige suave (papel académico)
- Texto: Marrón oscuro (tinta tradicional)
- Acentos: Azul profundo (autoridad), Terracota (énfasis)
- Propósito: Transmitir credibilidad y profundidad intelectual

**Layout Paradigm**:
- Dos columnas: narrativa + datos/visualizaciones
- Márgenes amplios tipo libro
- Separadores tipográficos (líneas, viñetas)

**Signature Elements**:
- Números grandes para probabilidades
- Citas destacadas en estilo editorial
- Viñetas decorativas y separadores

**Interaction Philosophy**:
- Transiciones suaves y elegantes
- Revelación gradual de información
- Enfoque en legibilidad

**Animation**:
- Entrada: fade suave
- Scroll: parallax sutil
- Números: animación de conteo elegante

**Typography System**:
- Display: Crimson Text (serif, elegante)
- Body: Lora (serif, legible)
- Data: JetBrains Mono (técnico, limpio)

---

## Decisión Final: Concepto 2 - Tablero Inteligencia Geopolítica

Se eligió el **Concepto 2** porque:
1. Refleja la naturaleza urgente y estratégica del contenido
2. Crea una experiencia inmersiva que mantiene la atención
3. El diseño cyberpunk es contemporáneo y memorable
4. Permite visualizar conexiones geopolíticas de forma intuitiva
5. Los acentos neon crean jerarquía visual clara sin ser caóticos
6. Las animaciones refuerzan la sensación de análisis en tiempo real

**Paleta Final**:
- Fondo: #0a0e27 (azul muy oscuro)
- Acentos primarios: #00ff88 (verde neón)
- Acentos secundarios: #00d4ff (cian)
- Alertas: #ff006e (magenta)
- Texto: #e0e0e0 (gris claro)

**Tipografía**:
- Títulos: Space Mono Bold
- Cuerpo: Roboto Mono
- Datos: IBM Plex Mono
